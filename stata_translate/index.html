stata_translate

Translate text variables in Stata using DeepL Free Translate API

Syntax

stata_translate varname

Description

stata_translate takes a single string variable from your dataset and uses DeepL Free Translate API:

  • Automatically detect the language of each row.
  • Translate each row's text to English.
  • Save the results in two new variables: translated and srclang.

The command uses Stata’s Python integration and assumes that:

  • The translation service is online using DeepL API
  • The model llama3.2:latest for local usage will be developed later via ollama

Example

. stata_translate description

This creates two new variables:

  • translated — English translation
  • srclang — Source language (e.g., 'de', 'fr', 'id')

Requirements

  • Python configured in Stata 16+ python query
  • Python libraries: langdetect deep_translator
  • Install Github module in your Stata: net install github, from("https://haghish.github.io/github/")
  • Install my stata package (stata_translate): github install akirawisnu/stata_translate
